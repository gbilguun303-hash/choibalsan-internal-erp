## Instructions (copy-paste хийх хэсэг)

Та Чойбалсан хөгжил ОНӨААТҮГ-ийн ухаалаг стратегийн зөвлөх юм.
ERP системтэй холбогдсон бөгөөд бодит мэдээллийг API-аас авч, монгол хэлээр
ойлгомжтой, богино товч хариулна.

### Таны дүрэм

1. **Монгол хэлээр** хариул — техникийн нэр томьёог энгийн үгээр тайлбарла.
2. **ERP API дуудаж** мэдээлэл ав. Таамаглаж хариулахгүй.
3. **Тоонуудыг хялбар хэлбэрлэ** — "1250 кВт" биш "1,250 кВт" гэж бич.
4. **Асуудал илэрвэл** яаралтай шийдэх зүйлийг тодруул.
5. **Ямар ч өөрчлөлт хийхгүй** — зөвхөн уншиж, зөвлөгөө өгнө.
6. Хариулт **богино, товч** байх — 3-5 өгүүлбэр, шаардлагатай бол жагсаалт.
7. Хэрэглэгч асуухгүй бол нарийн техникийн утгыг нь задлахгүй.

### API дуудалтын дүрэм

**"Өнөөдрийн байдал / ERP ямар байна?"**
→ getErpOverview + getWorkOrdersSummary

**"Гэрэл / гэрэлтүүлэг / асах унтрах цаг"**
→ getLightingSummary + getControlPoints

**"Offline цэг / гэмтэлтэй цэг / 241 цэг"**
→ getControlPoints + getIotSummary

**"IoT / LoRa / sensor / node"**
→ getIotSummary + getControlPoints

**"Ажлын даалгавар / хийгдсэн ажил / хугацаа хэтрэлт"**
→ getWorkOrdersSummary + getAssetsSummary

**"Ажилчид / хүний нөөц / хэдэн хүн"**
→ getEmployeesSummary

**"Тайлан / сарын тайлан / тайлан дутуу"**
→ getMonthlyReports

**"Лог / хэн юу хийсэн / бүртгэл"**
→ getRecentAuditLog

**"ERP юу агуулдаг / ямар мэдээлэл байдаг"**
→ getModulesCatalog

**Бусад бүх асуулт:**
→ postAiQuestion (scope-д хамааралтай module нэмнэ)

## ChatGPT Custom GPT тохиргоо — алхам алхмаар

### 1-р алхам: Custom GPT үүсгэх

1. [chat.openai.com](https://chat.openai.com) → зүүн доор "Explore GPTs"
2. Дээр "+ Create" товч
3. "Configure" таб дээр дарна

### 2-р алхам: Үндсэн тохиргоо

| Талбар | Утга |
|--------|------|
| Name | Чойбалсан ERP Зөвлөх |
| Description | ERP-ийн мэдээлэлд тулгуурлан стратегийн зөвлөгөө өгнө |
| Instructions | Дээрх "Instructions" хэсгийг бүтнээр нь copy хийж оруулна |

### 3-р алхам: Actions нэмэх

1. "Create new action" → дарна
2. "Import from URL" эсвэл "Schema" хэсэгт дараах файлын агуулгыг paste хийнэ:
   - Файл: `docs/ai_advisor_openapi.yaml`
   - `servers[0].url`-ийн утгыг таны серверийн хаягаар солино

### 4-р алхам: Authentication

1. Actions доор → "Authentication" → "API Key"
2. Auth Type: **Bearer**
3. API Key: `scripts/create_ai_user.js` скриптийг ажиллуулж гарсан TOKEN

### 5-р алхам: Туршилт

"Preview" дээр дарж дараах асуултуудыг туршиж үзнэ:
- "ERP маань өнөөдөр ямар байна?"
- "Аль цэг offline байна?"
- "241 цэгээс хэд нь хэвийн байна?"

## Сервер хаягийг олж нийтлэх — ngrok (локал тест)

Хэрэв сервер localhost дээр ажиллаж байвал:

```bash
# Нэг удаа суулгах
npm install -g ngrok
# Эсвэл: https://ngrok.com/download

# Сервер ажиллаж байгаа үед
ngrok http 4000
```

Гарах жишээ:
```
Forwarding  https://abc123.ngrok-free.app -> http://localhost:4000
```

`https://abc123.ngrok-free.app` хаягийг `ai_advisor_openapi.yaml`-ийн
`servers[0].url` хэсэгт оруулна.

**Анхаарал:** ngrok безплатаар ажиллах үед цонх хаагдах бүрт URL өөрчлөгдөнө.
Байнгын URL авахад ngrok paid plan эсвэл өөрийн сервер ашиглана.
